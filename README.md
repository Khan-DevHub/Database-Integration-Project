**Secure Admin User Management System
Overview**
This project is a robust, web-based Admin User Management System developed using PHP and MySQL. It features a modern, responsive interface built with Glassmorphism design principles. The system provides secure authentication and comprehensive CRUD (Create, Read, Update, Delete) functionality, enabling administrators to manage user accounts efficiently.

**Key Features**
- Secure Authentication: Login system with session-based access control.
- Role-Based Management: Differentiated access levels for system administrators.
- CRUD Operations: Seamless creation, retrieval, updates, and deletion of user records.

**Modern UI/UX:** A glassmorphism-inspired aesthetic featuring a clean, professional dark-mode dashboard.
